base

/src\sass\foundation\_base.scss
  • base
  • latest

リセットCSSのように、HTML要素に最低限のスタイルをあてます 要点は...

  • font-sizeはroot(html)に62.5%を指定しているため、1rem = 10pxです
  • 文字の単位はrem
  • ResetCSSはressを使用。グローバルセレクタ（*）でbox-sizing: border-box;指定済み
  • 見出しにはfont-feature-settings: "palt" 1;を指定。safariでバグがあるので、jsで出し分けしない場合は使用しないこと
